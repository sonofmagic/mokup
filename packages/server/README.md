# @mokup/server

English | [Chinese](./README.zh-CN.md)

## Overview

Server adapters for Mokup that integrate the runtime with popular Node frameworks and Workers. For detailed usage, see https://mokup.icebreaker.top.

## Entrypoints

- `@mokup/server`: runtime-agnostic fetch handler + types
- `@mokup/server/worker`: Worker-friendly entry
- `@mokup/server/node`: Node adapters and dev server helpers
