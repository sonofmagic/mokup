# Installation

## Requirements

- Node.js 20+
- pnpm (recommended)

## Packages

Dev tooling (Vite plugin + CLI):

::: code-group

```bash [pnpm]
pnpm add -D mokup
```

```bash [npm]
npm install -D mokup
```

```bash [yarn]
yarn add -D mokup
```

```bash [bun]
bun add -d mokup
```

:::

Runtime / server adapters (deploy or middleware):

::: code-group

```bash [pnpm]
pnpm add mokup
```

```bash [npm]
npm install mokup
```

```bash [yarn]
yarn add mokup
```

```bash [bun]
bun add mokup
```

:::

If you only need Vite dev mocks, install `mokup` as a dev dependency.
